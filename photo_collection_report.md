# photo_collection_report.md — 사진 1차 구축 작업 보고서

박효순님이 Numbers 파일(`atlas_image_input.numbers`)에 직접 채워주신 이미지를
실제 카드에 연결한 작업 결과다.

---

## 1. 입력 현황

- 받은 시트: 447행(사건 272 + 인물 148 + 세계사 27) 중 **146행에 이미지가 채워짐**
- 146건 중 **실사·기록 자료 91건(62%)**, **AI 생성 이미지 55건(38%, 박효순님 직접 제작)**
- 세계사 27건에는 이미지가 채워지지 않아 이번 작업 범위에 포함되지 않음

## 2. 매칭 결과

- 제목 기준 매칭 **146/146건 성공** (불일치 0건)
- 사건(type≠person) 83건 → 해당 사건 카드에 직접 연결
- 인물 63명 → 아래로 세분화:
  - **person 타입 카드가 있는 31명**: 그 인물이 등장하는 모든 person 카드에 동일 사진을 hero로 연결 (한 인물이 여러 연도 카드에 등장하면 전부 동일하게 연결 — `media-policy.md`의 "여러 카드 재사용 가능" 원칙)
  - **person 카드는 없지만 동명의 사건 카드가 있는 7명**(유관순·최수봉·강우규·김상옥·나석주·손기정·박재혁): 해당 사건에 연결. 이미 사건 자체의 사진이 있는 경우는 `gallery`에 인물 사진을 추가하고, 빈 경우는 `hero`로 사용
  - **연결할 카드가 없는 25명**: 이번 작업에서 보류 (4번 참고)

## 3. 실제 적용 결과

| 항목 | 수치 |
|---|---|
| content 필드가 추가된 카드 | 135개 |
| hero 이미지 | 135건 |
| gallery 이미지 | 6건 |
| 적용된 이미지 총합 | 141건 |
| AI 생성 이미지 | 63건 |
| 실사/기록 이미지 | 78건 |
| 저장 폴더 | `assets/images/entity/event/` 83개, `assets/images/entity/person/` 63개 |
| 원본 대비 용량 | 143MB → 22.3MB (webp 변환) |

**AI 생성 이미지는 모두 `credit: "AI 생성 이미지"`, `is_ai: true`로 표시했습니다.**
요청하신 "카드 하단에 작게 표시"는 데이터 레벨에서 이 두 필드로 준비해 두었고,
실제 화면에 작게 보이게 하는 렌더링(`renderer.js`의 `popupHtml()`)은 이번
세션에서 수정하지 않았습니다 — 5번 "남은 작업" 참고.

## 4. 보류된 인물 25명 (이유: 연결할 카드 없음)

다음 인물들은 person 타입 전용 카드도 없고, 이름과 일치하는 사건 카드도 없어
이번 세션에서는 연결하지 못했습니다. 이들은 다른 사건의 `people` 배열에만
이름이 언급되는 형태로 존재합니다.

| 인물 | AI 이미지 여부 |
|---|---|
| 김홍집, 고종, 홍영식, 박영효, 김옥균, 이토 히로부미, 박제순, 이근택, 이지용, 권중현, 전명운, 장인환, 이승훈, 조소앙, 이범석, 이동녕, 최현배 | 실사 |
| 서재필, 이준, 이시영, 신팔균, 김규식, 김동삼, 박차정, 조명하 | AI 생성 |

다음 세션 후속 작업 후보: 이 인물들을 위한 별도 person 카드를 새로 만들지,
아니면 다른 방식(예: 사건 카드의 `people` 배열에 사진을 직접 매핑하는 새로운
구조)으로 처리할지 결정이 필요합니다. **이번 세션에서는 사건 추가도, 인물 카드
신규 생성도 하지 않았습니다** — 받은 이미지를 기존 카드에만 연결했습니다.

## 5. 남은 작업 (다음 세션)

1. **`js/renderer.js`의 `popupHtml()`이 아직 `content` 필드를 읽지 않습니다.**
   이번 세션은 데이터에 `content.hero`/`content.gallery`를 채우는 작업까지만
   했고, 화면에 실제로 표시되게 하려면 렌더링 함수 수정이 필요합니다(이전
   `content_model.md`/`media-policy.md` 설계를 그대로 적용하면 됩니다).
2. **AI 표기를 카드에 작게 보이게 하는 CSS/HTML**도 같은 이유로 아직 미반영
   — 데이터(`is_ai`, `credit`)는 준비됐으니 화면 표시만 남았습니다.
3. **보류된 25명**의 처리 방향 결정.
4. **세계사 27건**은 이번 시트에 이미지가 없어 다음 라운드로 남음.
5. **나머지 197건의 사건 + 84건의 인물 카드**는 아직 사진이 없는 상태입니다
   (343건 중 135건만 이번에 채워짐). 1차 시트의 나머지 빈 칸도 계속 채워주시면
   같은 방식으로 반영하겠습니다.
