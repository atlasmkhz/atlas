## ⚠ 기술 부채 (Technical Debt) — "구조 준비 완료" ≠ "기능 완료"

## 구조적 변경 (엔진)

- **데이터 구조 분리**: 단일 `EVENTS` 배열 → 연도별 `EVENTS_1920`~`EVENTS_1945` 분리 → `DATA[year]` 객체로 자동 조립.
- **ID 기반 연결로 전환**: `connections`가 한글 제목 문자열 참조이던 것을 전부 `id` 참조로 마이그레이션. 88개(이후 197개) 사건 전체 재작업.
- **점 생명주기(visible_from/visible_until) 도입**: 연도 이동 시 사건이 나타나고, 흐려지고, 사라지는 구조. 이전에는 해당 연도 사건만 단순 교체.
- **누적 감쇠 로직 추가 및 보정**: 3년 지난 배경(priority:2) 사건은 자동 소멸하되, `policy`/`migration` 타입은 시대 배경으로서 예외 처리.
- **마커 겹침 자동 해소(declutter) 구현**: 경성처럼 사건이 밀집한 지역에서 마커가 겹쳐 클릭이 안 되는 문제를 좌표 미세조정이 아닌 화면 좌표 기반 자동 펼침으로 해결.
- **연결선 시각화 + dedupe**: `connections` 데이터를 실제 점선으로 렌더링. 좌표 기반 Set으로 양방향 중복선 제거.
- **干支 자동 계산**: 연도별 하드코딩 → 1864년 갑자년 기준 60갑자 순환 계산식으로 전환.
- **검증기 도입**: `validateConnections()`, `auditYear()`, `auditNarrative()` 등 → 이후 `validateDuplicateIds`, `validateCoordinates`, `validateLifecycle`, `validateOrphans`, `validateCircularRefs`, `validateMissingYears` 추가.
- **importance/줌 레벨 연동**: 사건 수가 늘어도 화면이 과밀해지지 않도록 global/regional/local 3단계 표시 밀도 도입.
- **시대 색조 전환을 키프레임 보간 방식으로 개선**: 연도 점프 시 색이 튀던 문제 → 5개 키프레임 사이 선형 보간으로 자연스럽게 전환.

## 모바일/안정성 버그 수정

- **지도가 전혀 안 뜨는 문제**: Leaflet 라이브러리 로딩 소스를 unpkg.com → jsdelivr로 변경. 동시에 로드 실패 시 안내 메시지를 띄우는 가드 추가.
- **지도를 가로로 한 바퀴 돌리면 점이 전부 사라지는 문제**: `maxBounds` + `worldCopyJump:false` + 타일 `noWrap:true` 조합으로 해결.
- **JS 문법 깨짐 (Script error)**: 영어 제목에 아포스트로피(`'`)를 직접 사용해 문자열 이스케이프가 깨진 사례 2건 발견 및 수정. 이후 영어 제목 작성 시 아포스트로피 사용을 피하는 규칙으로 정착.
- **팝업이 화면 밖으로 잘리는 문제**: `autoPan:true`, `autoPanPadding`, `keepInView:true` 옵션 추가.
- **iOS Safari에서 지도 컨테이너가 0×0으로 계산되는 문제**: `window.onload` → `requestAnimationFrame` → `setTimeout` 순으로 단계적 `invalidateSize()` 호출.

## 지리적 정확성 수정

- **동해 표기 문제**: 배경 타일(OpenStreetMap 계열)이 "Sea of Japan"을 표기하는 문제 발견. 1차로 단순 라벨 추가만으로는 원본 표기가 가려지지 않아, 위경도 고정 사각형으로 해당 영역을 가리고 그 위에 "동해 East Sea" 라벨을 재배치하는 방식으로 수정했다. **단, 이 수정은 스크린샷 1장을 기준으로 좌표를 추정한 것이며, 모든 줌 레벨에서 검증되지 않았다 — 위 기술 부채 표 참고. "해결"이 아니라 "1차 조치"로 기록한다.**
- **독도 누락**: 배경 타일에 독도가 표시되지 않는 문제. 정확한 좌표(37.2417°N, 131.8639°E)에 별도 마커 추가. 이 마커는 고정 좌표라 가리개와 달리 줌 레벨에 무관하게 항상 정상 표시됨 (검증됨).
- **경성(서울) 내 사건 좌표 밀집 문제**: 1920~1927년 사건 중 다수가 종로 일대의 거의 같은 좌표에 몰려 확대해도 구분이 안 되는 문제. 각 사건의 실제 역사적 위치(서대문형무소, 경복궁 앞, 을지로, 창덕궁, 정동, 신촌 등)로 재배치.

## 데이터 보강 (역사적 정확성/균형)

- **훈춘사건 서술 층위화**: "일본이 마적단을 매수한 조작 사건"이라는 단정적 서술을 "다수 연구는 조작극으로 보지만 사료에 따라 이견이 있다"는 식으로 수정.
- **조선혁명선언/권기옥의 visible_until 오류 수정**: 사건의 사상적 영향력과 "그 장소에 사람/사물이 실제로 머문 기간"을 혼동해 visible_until을 과도하게 길게(1945년까지) 잡았던 오류 발견 및 수정. 이 사례가 schema.md의 visible_until 작성 원칙에 반영됨.
- **사건 유형 분류 정리**: 대한통의부·참의부·정의부·신민부 등 "조직 결성" 사건이 `battle` 타입으로 잘못 분류되어 있던 것을 발견, 신규 타입 `organization`을 만들어 7건 재분류.
- **누락 인물/사건 보강**: 회사령 폐지, 박헌영, 여운형, 강상호(형평운동), 정칠성(여성운동), 주기철(신사참배 저항) 등 엘리트 독립운동 중심으로 쏠려 있던 데이터에 경제·사회·종교 축 보강.
- **1925년 민중운동 공백 보강**: 신의주사건(제1차 조선공산당 검거), 을축년 대홍수 추가.
- **연도별 서사 균형 검증(`auditNarrative`)으로 발견한 약점 보강**: 1920년 고립 사건 비율 59%→16%로 개선 (역사적으로 타당한 연결만 추가, 억지 연결 지양).
- **식민지 폭력 축 보강(2026-06)**: 1910~1918년 무단통치기에 massacre/plot 유형 사건이
  1건뿐이었던 공백을 헌병경찰제·조선태형령·제1차 조선교육령(이상 1910~1912)·국민정신
  총동원조선연맹(1938) 4건으로 메움. 노동·여성 생활사 인물 강주룡 추가. 토지조사사업·
  회사령·원산총파업 등 기존 사건에 사실/해석/통계 보강. `impact_type`/`violence_level`/
  `victim_group` 필드를 schema.md에 신규 정의하고 8건에 시범 적용(전체 일괄 적용은 후속
  작업). 상세 내역은 `CHANGELOG_2.md`, `report_2.md` 참고.

## 데이터 확장 (연도별)

1920년(32→34건)부터 1945년(6건)까지 26개 연도 순차 작성. 각 연도 작업 시 동일한 워크플로우 적용: 웹 검색으로 핵심 사실 확인 → 데이터 작성(ID/connections/tags/sources 포함) → HTML(이후 분리된 파일)에 통합 → 연결 무결성·중복 ID·문법 검증 → 시대(ERA) 서술 갱신.
