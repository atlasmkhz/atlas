# 현대 지도(1994~2025) 기획안

## 0. 왜 이 문서부터인가

연도당 5개씩, 32년(1994~2025)이면 160개 사건입니다. 그중 상당수가
현재진행형 재판·생존 정치인 관련이라 — 특히 2016년 이후는 사실관계
하나하나를 검증해야 합니다. 한 번에 160개를 다 쓰면 검증 없이 기억에
의존해 쓰게 되는 항목이 반드시 생기고, 그게 하필 제일 민감한 부분
(현재진행 중인 재판, 의혹의 정확한 단계)일 가능성이 높습니다. 그래서
설계 → 검증된 뼈대 → 시기별 배치 순서로 갑니다.

## 1. 범위

- **기간**: 1994 ~ 2025 (이재명 정부 출범, 2025.6.4)
- **연도당 대형 사건 약 5개** — 정치/외교, 대형참사, 인물(의혹·재판 포함)
  을 고루 포함
- **정권 교체 구간(PHASES) 표시** — 근현대(modern2) 지도의 "정권 띠"
  기능을 그대로 재사용

## 3. 데이터 스키마 확장

기존 카드 스키마(`data/*.js`)에 두 필드를 추가합니다. 근현대(modern2)
지도까지 소급 적용할 필요는 없고, 새로 만드는 현대 지도부터 표준으로
씁니다.

```js
{
  // ...기존 필드(id/type/title_ko/summary_ko/sources 등) 그대로...

  party: '국민의힘',        // 사건 발생 당시 소속 정당. 정당명이 바뀐
                             // 경우 반드시 "그 시점" 이름을 쓴다(예:
                             // 신한국당→한나라당→새누리당→국민의힘처럼
                             // 개칭·분당·합당 이력이 많은 계열은 특히
                             // 주의). 정당과 무관한 사건(참사 등)은
                             // party: null.

  allegation_status: null,  // 의혹·재판 관련 사건에만 사용. 아래 값 중 하나:
                             // 'alleged'(의혹 제기 단계, 기소 전)
                             // 'indicted'(기소)
                             // 'trial_1st' / 'trial_2nd' / 'trial_final'(1심/2심/대법원 확정)
                             // 'convicted'(유죄 확정) / 'acquitted'(무죄 확정)
                             // 정당/재판과 무관하면 null.
}
```

`type`에 `'disaster'`(대형참사)를 새 값으로 추가합니다. 지금까지
`massacre`(국가폭력에 의한 학살)와는 인과관계가 다른 범주라 구분합니다
— 세월호처럼 국가 책임 논란이 있는 참사도 "사고/재난"과 "책임 소재
논란"을 본문에서 구분해 서술하고, type 자체는 `disaster`로 통일합니다.

## 4. 편집 원칙 (이 지도에서 특히 중요)

1. **의혹은 의혹으로, 확정은 확정으로.** `allegation_status`가 `alleged`
   단계인 사안을 마치 유죄가 확정된 것처럼 서술하지 않습니다. 반대로
   확정판결이 난 사안을 "의혹"으로 얼버무리지도 않습니다. 진행 중인
   사건은 본문에 "이 글 작성 시점 기준"이라는 표현을 넣어, 나중에
   재판 결과가 바뀌어도 무엇을 기준으로 쓴 서술인지 알 수 있게 합니다.
2. **팩트(공개 활동)와 의혹(범죄 혐의)을 구분합니다.** 나경원 의원의
   자위대 행사 참석처럼 "실제로 있었던 공개된 활동"은 사실관계 서술로
   다루고, 형사 혐의가 걸린 사안과는 다른 톤으로 씁니다 — 전자는
   "무엇을 했는가"의 기록이고, 후자는 "유죄인가 아닌가"가 핵심이라
   같은 방식으로 쓰면 안 됩니다.
3. **정당 소속은 사건 시점 기준.** "당시 한나라당 의원" 식으로, 지금
   소속이 아니라 사건 발생 시점의 소속을 명시합니다. 이후 탈당·이적·
   제명이 있었다면 그것도 후속 사실로 남깁니다.
4. **생존 인물 서술은 근거 표기를 더 촘촘히.** 이미 카드 스키마에
   `sources`가 있지만, 생존 인물·진행 중인 재판 관련 카드는 특히
   기사 날짜·판결 선고일까지 구체적으로 남겨서 "언제 기준의 서술인지"
   추적 가능하게 합니다.
5. **정치적으로 여전히 다투는 사안(계엄·탄핵 등)은 공식 판단(헌재
   결정문·법원 판결·정부 공식 발표) 중심으로 쓰고, 논쟁적 해석은
   "~라는 평가/비판이 있다"처럼 주체를 밝혀 서술합니다** — 이건 이미
   ATLAS가 5·18·역사왜곡 콘텐츠에서 쓰고 있는 방식과 동일합니다.

이 중 근현대(1945~1993) 지도 범위 안에 있는 것은 전부 이번 "현대
지도" 작업과 별개로 **기존 modern2 데이터에 소급 추가**할 수 있습니다.
(1876~1945 범위에 드는 참사가 있는지도 별도로 점검이 필요합니다 —
지금은 확인 안 함.)

> 나무위키 검색 중 "한성호=김기춘이 세월호를 덮으려 기획" 같은 음모론도
> 나왔는데, 이건 근거 없는 주장이라 배제했습니다(언론 보도량 자체가
> 적어 "정치적 이슈 덮기"로 보기 어렵다는 반박까지 같은 문서에 있음).
> 사실만 남기고 음모론은 자료실에서도, 지도 카드에서도 쓰지 않습니다.

## 6. 1차 초안 배치 — 1994~1997 (김영삼 정부, 스켈레톤)

전체 32년을 한 번에 검증 없이 쓰지 않기 위해, 상대적으로 사실관계가
안정된 이 4년부터 스켈레톤(제목·유형·한 줄 설명)으로 작성했습니다.
이후 정권 단위로 이어가겠습니다. 이 단계는 아직 카드 전체 문서(summary_ko
전문·sources 등)가 아니라 "이 정도 사건을 이 정도 비중으로 다루자"는
목록 검토용입니다 — 순서·비중·누락 여부를 먼저 확인해주세요.

### 1994
1. **성수대교 붕괴** (10.21, disaster) — 32명 사망. 부실시공·관리소홀.
2. **김일성 사망** (7.8, political) — 북한, 김정일 체제로.
3. **지존파 사건** (9~10월, 사회) — 연쇄살인, 사회적 충격.
4. **북한 核 위기(제네바 합의)** (10월, 외교) — 1차 북핵위기 봉합.
5. **전국민 의료보험 확대·금융실명제 정착기** — 필요시 다른 항목과 교체 검토.

### 1995
1. **삼풍백화점 붕괴** (6.29, disaster) — 502명 사망, 한국 최악의 인재.
2. **대구 지하철 가스폭발** (4.28, disaster) — 101명 사망.
3. **전두환·노태우 구속** (12월, political) — 5·18 특별법, 반란·내란죄.
4. **지방자치제 전면 부활** (6월, political) — 30년 만의 지방선거.
5. **역사바로세우기 — 조선총독부 건물 철거 착수** (political) — 자료실
   「역사왜곡에 맞서다」와 연결 가능성 있음(반대 성격의 사건).

### 1996
1. **전두환·노태우 1심 선고** (8월, political) — 무기징역/징역형 (당시
   민주자유당 계열, 자세한 소속은 검증 필요).
2. **연세대 사태(한총련 통일축전 충돌)** (8월, 사회) — 대규모 공권력 투입.
3. **강릉 무장공비 침투사건** (9월, 안보) — 잠수함 좌초, 대규모 소탕작전.
4. **OECD 가입** (10월, 외교) — 선진국 클럽 가입, 이후 외환위기와 대비.
5. **노동법·안기부법 날치기 처리** (12월, political) — 총파업 촉발.

### 1997
1. **IMF 외환위기** (11~12월, 경제/정치) — 가장 굵직한 사건, 상세 서술 필요.
2. **한보 사태** (1월, political) — 정경유착 스캔들, 김영삼 차남 연루 의혹.
3. **대한항공 801편 괌 추락** (8.6, disaster) — 228명 사망.
4. **황장엽 망명** (2월, political) — 최고위 탈북.
5. **15대 대선 — 김대중 당선(최초 여야 정권교체)** (12월, political).

## 7. 다음 단계

1. 위 1994~1997 스켈레톤을 검토해주세요 — 순서·비중·빠진 사건 지적해
   주시면 그다음에 정식 카드(요약문·출처·people·party 필드 포함)로
   완성하겠습니다.
2. 이후 배치는 정권 단위로 이어가되, **2016년 이후(박근혜 탄핵~현재)는
   항목마다 검색으로 개별 검증**하면서 진행하겠습니다 — 특히 의혹의
   현재 재판 단계, 정당 소속 변경 이력은 지금 시점 기준으로 다시
   확인해야 정확합니다.
3. 기술 작업(새 서브사이트 `maps/contemporary/` 골격, PHASES 배열,
   `party`/`allegation_status` 필드 추가, disaster 타입 마커 아이콘)은
   콘텐츠 뼈대가 어느 정도 잡힌 뒤 한 번에 진행하는 게 효율적입니다 —
   지금 코드부터 만들면 데이터 구조가 바뀔 때마다 다시 손대야 합니다.
4. 근현대 지도 참사 보강(5번 항목)은 콘텐츠 검증이 거의 끝난 상태라
   원하시면 바로 카드로 만들어 넣을 수 있습니다.
