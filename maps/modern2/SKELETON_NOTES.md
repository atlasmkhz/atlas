# 근현대(1945~1993) 지도 — 골격 단계 작업 요약

이번 작업은 **데이터 없이 연도 레이아웃만** 잡는 단계다. 실제 사건/인물 데이터는
다음 작업에서 채운다.

## 무엇을 만들었나

- `maps/modern2/` — 근현대 지도의 독립 디렉토리. 근대 지도(`/`)와 같은 패턴
  (index.html + css/style.css + js/*.js + data/*.js)을 그대로 복제했다.
- `data/1945.js` ~ `data/1993.js` — 49개 연도 파일을 전부 빈 배열
  (`const EVENTS_YYYY = [];`)로 생성했다. 근대 지도가 "사건이 있는 해만 파일을
  만드는" 방식이었던 것과 달리, 이번엔 흥두목님 요청대로 전 연도를 일단 다
  비워서 만들어뒀다.
- 상단 포털 내비게이션(`nav.js`)에서 "근현대" 카드가 입장 가능(`ready:true`)으로
  바뀌었고, 근대 지도(`/index.html`)와 근현대 지도(`/maps/modern2/index.html`)
  양쪽에서 서로를 오갈 수 있다.

## 연도 경계

- **근대**: 1876~1945 (기존 그대로, 변경 없음)
- **근현대**: 1945~1993
- 1945년은 두 지도 모두에 존재하는 "다리 해"다 — 근대 지도는 광복으로
  끝맺고, 근현대 지도는 광복에서 시작한다. 같은 해를 서로 다른 결로 다룬다.

## 데이터 없이도 안전하게 동작하도록 손본 부분

근대 지도의 `js/app.js`, `js/timeline.js`를 그대로 복사해 오면 **1946~1993년
전체가 "조선총독부 통치하"로 잘못 표시되는 버그**가 있었다(연도 분기가
1876~1945 전용으로 하드코딩돼 있었기 때문). 이번에 다음을 고쳤다:

- `app.js`의 `DATA` 조립 배열을 `EVENTS_1945`~`EVENTS_1993`으로 교체.
  `INITIAL_YEAR`도 1945로 맞춤.
- `timeline.js`의 `ERA`(연도별 시대 설명) 객체를 비우고 1945 한 항목만
  남김. 빈 연도는 `ERA[year] || {text:'', desc:''}` 가드로 안전하게
  빈 문자열을 반환한다.
- `timeline.js`의 `PHASES`(시대 국면 띠)를 빈 배열로 비움 — 한국전쟁,
  유신, 민주화 항쟁 같은 국면 구분은 데이터를 넣을 때 설계할 것.
- `timeline.js`의 `ERA_TINT_KEYS`(연도별 화면 색조)를 1945→1993
  단순 보간으로 임시 교체.
- `timeline.js`의 국체 표시(occTag) 분기를 근현대에 맞게 재작성:
  해방(1945) → 미군정·소군정기(1946~1947) → 대한민국(1948~1993).

## 이번에 일부러 안 한 것 (다음 작업 범위)

- `world_events.js`, `world_context.js`, `places/places_min.js`는 아직
  없다. `renderer.js`가 `typeof WORLD_EVENTS === 'undefined'` 가드로
  안전하게 건너뛰므로 에러는 없지만, 세계사 레이어와 동해/독도 표기가
  필요해지면 근대 지도와 같은 패턴으로 추가해야 한다.
- `PHASES`(시대 국면), 정확한 `ERA_TINT_KEYS`(연도별 색조), 인물/사건
  이미지(`assets/images/`)는 전부 비어 있다.
- og:image, canonical URL은 근대 지도 값을 임시로 재사용한 placeholder다.
  실제 배포 경로가 확정되면 교체할 것.

## 검증 방법

이 환경은 Playwright의 실제 브라우저 구동이 네트워크 정책상 막혀 있어,
`node --check`(문법)와 `jsdom`(DOM 시뮬레이션)으로 검증했다:

- 49개 데이터 파일 전체 문법 통과, 빈 배열 49개 정상 로드 확인.
- `app.js`의 `DATA` 조립 로직이 빈 데이터에서 안전하게 0개 키를 반환.
- `getGanji(1945)`=乙酉年, `getGanji(1993)`=癸酉年 등 干支 계산 정상.
- `getEraTint`, `getPhase`가 1945~1993 전체 연도에 대해 에러 없이 동작.
- 근대 ↔ 근현대 양쪽 `nav.js`에서 서로의 카드가 "입장 가능"으로 뜨고,
  클릭 시 정상적으로 상대 페이지 경로(`../../`, `.`)를 가리키는 것을
  jsdom으로 직접 클릭 이벤트를 발생시켜 확인.
