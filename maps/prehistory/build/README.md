# SEO 정적 페이지 생성기 — 아직 없음

이 지도(선사시대, 신화시대~고조선)는 아직 시드 데이터(블록 6개에
총 14건)만 있는 스캐폴드 단계라 SEO 정적 페이지 생성기를 만들지
않았습니다. 데이터 구조는 고대(ancient) 지도와 같은 "블록 파일
하나씩" 방식(data/blockN.js가 EVENTS_BLOCKN을 export)입니다.

콘텐츠가 어느 정도 쌓이면, 고대 지도 것을 참고해 이 구조에 맞는 새
generate_event_pages.py를 작성하고 통합 sitemap.xml에도 반영해야
합니다 — 지금 시드 카드 14개만으로는 만들 가치가 낮아 보류.

또한 "환단고기와 대종교" 자료실 시리즈(4편 예정: 문헌 개관 / 상고사
서사 상세 / 위서 논쟁 근거 / 대종교와 독립운동)는 아직 만들지
않았습니다 — 지도 카드에서는 개관만 다루고, 상세 서사·위서 논쟁
근거는 이 시리즈가 만들어지면 연결할 계획입니다.
